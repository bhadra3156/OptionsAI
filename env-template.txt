# OptionsAI — Environment Variables Template

# HOW TO USE THIS FILE:
# 1. In your project root folder, create a file called exactly: .env.local
# 2. Copy everything below into it
# 3. Replace every "xxxxx" with your real API keys

# 5. Add these same variables to Vercel dashboard before deploying
